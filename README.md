# Ozip2Img

Converts Oppo OZIP files to Ext4 images by extracting and creating

## Requirements

```bash
Java Runtime Environment (JRE)
mke2fs
python3
OZIP File
```

## Usage

Clone the repository

```bash
git clone https://github.com/deep-sengupta/Ozip2Img.git
cd Ozip2Img
```

 Run Ozip2Img
```bash
chmod +x ./ozip2img.sh
./ozip2img.sh

python3 ozip2img.py
```
